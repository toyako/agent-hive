# Changelog

## 1.0.0

First public release.

Features:
- Multi-Runtime Orchestration (Claude, Codex, Hermes, OpenClaw)
- Unified Entry: `hive run --task "..."`
- Runtime Benchmark & Intelligence
- Observability (traces, metrics, conversations)
- Review & Revision Loop
- Visual Dashboard
- Runtime Discovery
- Setup Wizard (`hive setup`)
- Self Hosting (doctor, health, version)
