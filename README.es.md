# 🐝 Agent Hive

[English](README.md) | [中文](README.zh-CN.md) | [日本語](README.ja.md) | **Español**

**Una tarea. Múltiples mentes AI.**

Claude escribe. GPT revisa. Gemini planifica.

Agent Hive permite que múltiples modelos AI colaboren automáticamente en la misma tarea.

```bash
npm install -g agenthive
hive setup
hive "construir una landing page SaaS"
```

## Inicio Rápido

```bash
npm install -g agenthive
hive setup
hive "construir una API CRUD de usuarios"
```

## Comandos

| Comando | Descripción |
|---|---|
| `hive` | Modo interactivo |
| `hive "tarea"` | Ejecutar tarea |
| `hive setup` | Configurar proveedor |
| `hive doctor` | Verificar salud |
