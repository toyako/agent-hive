# 🐝 Agent Hive

[English](README.md) | [中文](README.zh-CN.md) | **日本語** | [Español](README.es.md)

**一つのタスク。複数のAIマインド。**

Claudeが書き、GPTがレビューし、Geminiが計画する。

Agent Hiveは複数のAIモデルが同じタスクに自動的に協力します。

```bash
npm install -g agenthive
hive setup
hive "SaaSランディングページを作成"
```

## クイックスタート

```bash
npm install -g agenthive
hive setup
hive "ユーザーCRUD APIを作成"
```

## コマンド

| コマンド | 説明 |
|---|---|
| `hive` | インタラクティブモード |
| `hive "タスク"` | タスク実行 |
| `hive setup` | プロバイダー設定 |
| `hive doctor` | ヘルスチェック |
