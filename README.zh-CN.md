# 🐝 Agent Hive

[English](README.md) | **中文** | [日本語](README.ja.md) | [Español](README.es.md)

**一个任务。多个 AI 大脑。**

Claude 写代码。GPT 审查。Gemini 规划。

Agent Hive 让多个 AI 模型自动协作完成同一个任务。

```bash
npm install -g agenthive
hive setup
hive "构建一个 SaaS 落地页"
```

## 快速开始

```bash
npm install -g agenthive
hive setup
hive "构建一个用户 CRUD API"
```

## 为什么选择 Agent Hive？

| 单代理 | Agent Hive |
|---|---|
| 写代码 + 自我审查 | 写代码 + 独立审查 |
| 容易遗漏自己的错误 | 第二双眼睛 |
| 单一模型视角 | 多模型协作 |

## 命令

| 命令 | 说明 |
|---|---|
| `hive` | 交互模式 |
| `hive "任务"` | 运行任务 |
| `hive setup` | 配置供应商 |
| `hive doctor` | 健康检查 |
| `hive memory list` | 查看记忆 |
| `hive project list` | 查看项目 |
